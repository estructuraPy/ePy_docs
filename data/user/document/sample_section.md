# Sección Externa (Markdown)

Este contenido proviene de un archivo **Markdown externo** (.md).

## Subsección de ejemplo

Puedes incluir:

- Listas con viñetas
- **Texto en negrita**
- *Texto en cursiva*
- `Código inline`

### Código de ejemplo

```python
def calcular_momento(fuerza, distancia):
    """Calcula el momento flector."""
    return fuerza * distancia
```

> **Nota:** Los archivos `.md` se integran directamente al flujo del documento.
