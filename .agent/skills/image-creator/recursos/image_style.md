# Guía de Estilo: Ilustración Técnica de Ingeniería Civil

Para asegurar la consistencia visual en todo el libro, todas las imágenes generadas por IA deben heredar los siguientes parámetros estéticos:

## Prompt de Estilo Base (System Prompt para Generación)
> "Ilustración técnica de ingeniería civil de alta calidad, estilo realista pero con trazos limpios de boceto técnico. Paleta de colores suaves y profesionales (gris concreto, azul acero, beige papel, amarillo seguridad). Iluminación de estudio suave para resaltar texturas y volúmenes. Fondo neutro limpio. Incluir etiquetas de texto en español con líneas indicadoras finas."

## Consideraciones Técnicas
1. **Claridad**: Los elementos estructurales (vigas, columnas, armaduras, suelos) deben ser claramente distinguibles.
2. **Texturas**: Resaltar el acabado del concreto, el brillo del acero y la porosidad del papel de plano cuando sea necesario.
3. **Optimización**: Preferir formatos eficientes para web/pdf que mantengan la nitidez de las líneas finas sin archivos de tamaño excesivo.
4. **Idioma**: Todas las etiquetas de texto internas de la imagen deben estar en **español**.
